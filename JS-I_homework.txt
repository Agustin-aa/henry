VARIABLE:
Una variable es una cosa que puede tomar un valor en un momento dado, y en otro momento puede tener otro valor.
Es decir, es algo que puede no valer siempre lo mismo.

STRINGS:
Una string es un conjunto de letras (y/o numeros y/o simbolos) que se encierran entre comillas para representar texto en un lenguaje de programacion.  

FUNCIONES
Una funcion es una "maquina" que recibe elementos de un determinado tipo (argumentos) trabaja con ellos (según como esté definida), y produce elementos de salida.
Para utilizar estos elementos que devuelve la funcion, debemos usar el comando return, ya que si no lo hacemos esos elementos quedaran dentro de la funcion 
sin poder utilizarase.

DECLARACIONES if:
Estas declaraciones se utilizan para decidir qué camino tomar, según se cumpla una condición o no, en un algoritmo.
Se utilizan simbolos de comparacion, >, >= <, <=, == !==, para evaluar la condición.

VALORES BOOLEANOS:
Los valores booleanos son expresiones que sirven para responder preguntas. Por ejemplo, el resultado de la expresion 2 + 2 == 4
es true, y el resultado de la expresion 3 * 5 == 0 es false. Los valores con los que se trabajara en JS son true y false, que representan
verdadero y falso respectivamente.
